# Documentation Archive

This directory holds completed internal-process records — PR plans, retirement records, and other
historical workflow detail — kept for accuracy rather than as active guidance. See
[10 — Migration and cleanup plan](../documentation-upgrade/10-migration-and-cleanup-plan.md)'s
"Archive candidates" section for the criteria used to decide what belongs here.

Current reference material (architecture, governance, component catalogs, validation checklists)
stays in `docs/design-system/` and `docs/documentation-upgrade/`. Nothing in this archive is the
current source of truth for anything — each archived file links to whatever superseded it.
